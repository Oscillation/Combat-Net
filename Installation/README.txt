Extract 'Combat-Net.rar' to your root directory.
You should have the following directories: 
1. "Combat-Net\Client\"
2. "Combat-Net\Server\"

Extract 'prerequisites.rar' to your root directory.

Run 'oalinst.exe' and follow the installation instructions.
Run 'vcredist_x86.exe' and follow the installation instructions.

You now have all of the prerequisites required in order to run Combat-Net, and can delete the extracted files from 'prerequisites.rar'.
