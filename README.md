Particle File Structure
-----------------------

Text written in *cursive* represents an example value.

* Time-Range(seconds) : *0.5 1*
* R-Range : *0 255*
* G-Range : *0 255*
* B-Range : *0 255*
* Speed(pixels/frame) : *25*
* Speed-Range() : *0 5*
